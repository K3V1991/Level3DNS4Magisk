<h1 align="center"><b>Level3DNS4Magisk</b></h1>
<h4 align="center">This Module makes your Device to use Level3 DNS Servers</h4>
<br />

## Level3 DNS Servers:
* 209.244.0.3
* 209.244.0.4
<br />

## Caution:
It is not guaranteed to work on Cellular Networks on all Devices
<br />

## Requirements:
Magisk installed (v17+)
<br />
